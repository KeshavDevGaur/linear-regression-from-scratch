# 📈 Linear Regression from Scratch

## 🔍 Overview

This project implements Linear Regression using NumPy without using any ML libraries. It demonstrates how machines learn patterns from data using Gradient Descent.

---

## 🚀 Features

* Implemented Linear Regression from scratch
* Used Gradient Descent for optimization
* Applied Normalization for faster convergence
* Visualized results using Matplotlib

---

## 📊 Dataset

* Input: House Size
* Output: House Price

---

## ⚙️ Tech Stack

* Python
* NumPy
* Matplotlib

---

## 📈 Results

* Model successfully learns the relationship between size and price
* Cost decreases over iterations

---

## 💡 Key Learnings

* Gradient Descent
* Cost Function (MSE)
* Model Optimization
